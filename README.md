# 002 3の倍数
ユーザーに入力させた数字が3の倍数かどうかを判定する。 3の倍数の場合は"3の倍数です"、違う場合は"3の倍数ではありません"と表示する。
- 実例1
  - 数字を入力してください
    - 入力:3
    - 出力:"3の倍数です"
  - 計算を終了します。
   
- 実例2
  - 数字を入力してください
    - 入力:3.3
    - 出力:"3の倍数です"
   - 計算を終了します。
   
- 実例3
